===============
Migration Guide
===============

This migration guide provides information on how to transition from older versions of CoRelAy to the latest version. It is intended for users who have previously used CoRelAy and are now upgrading to a newer version. The guide outlines the changes made in the new version and provides instructions on how to adapt your code accordingly.
